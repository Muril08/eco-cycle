## EcoCycle

Sistema de composteira automatizada com foco em sustentabilidade e agricultura urbana.
O projeto utiliza Arduino para monitoramento ambiental e um dashboard web para visualização dos dados em tempo real.

## Problema
O descarte incorreto de resíduos orgânicos contribui para impactos ambientais negativos.
Muitas pessoas não têm acesso a soluções práticas e automatizadas para compostagem urbana.

## Solução
O EcoCycle propõe uma composteira automatizada que:
- Monitora condições ambientais
- Auxilia no controle do processo de compostagem
- Incentiva práticas sustentáveis em ambientes urbanos

## Funcionalidades
- Monitoramento de temperatura e umidade
- Integração com sensores ambientais
- Visualização de dados em dashboard web
- Sistema baseado em Arduino

## Tecnologias Utilizadas
- Arduino (C++)
- Sensores ambientais
- HTML, CSS e JavaScript
- Dashboard Web
- Vercel (deploy)
